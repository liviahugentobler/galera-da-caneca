Etapa 1 – Planejamento e Protótipo do Sistema

Nesta primeira etapa do Projeto Integrador, foi realizada a retomada da ideia do sistema Galera da Caneca, com foco no planejamento da estrutura do software e na validação inicial do seu funcionamento.

Inicialmente, foi elaborado um diagrama simples de classes, identificando as principais entidades do sistema, como usuários, produtos, pedidos e pagamentos. As classes foram definidas com base nos requisitos funcionais, utilizando conceitos de orientação a objetos, como herança e encapsulamento, para representar corretamente os diferentes tipos de usuários (cliente, funcionário e gerente).

Em seguida, foi criado um projeto Java Application utilizando Ant no NetBeans, com organização em pacotes, separando as classes de modelo e a classe principal. As classes foram implementadas apenas com sua estrutura básica, contendo atributos e métodos principais, sem regras de negócio completas, pois o objetivo desta etapa é estrutural.

Por fim, foi desenvolvido um protótipo de funcionalidade no método main(), simulando o fluxo do sistema por meio da criação de objetos e chamadas de métodos. O protótipo demonstra ações como cadastro de produto, realização de pedido e atualização de status, utilizando mensagens no console para representar o funcionamento do sistema.

A integração com banco de dados MySQL e a implementação completa das funcionalidades serão realizadas nas próximas etapas do projeto.
